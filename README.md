# Introdução

Bem-vindo ao guia do usuário do aplicativo Starken Insurance. Este documento fornece uma visão detalhada de todas as funcionalidades e seções da plataforma, projetado para ajudar os usuários a navegar e utilizar o sistema de forma eficaz.

A Starken Insurance é uma plataforma completa para gerenciamento de cotações de seguros, permitindo o controle de todo o ciclo de vida de uma apólice, desde a prospecção até o pós-venda. O sistema foi desenvolvido para otimizar o fluxo de trabalho de corretoras, seguradoras e colaboradores, centralizando informações e automatizando processos.

## Estrutura da Documentação

Esta documentação está organizada nas seguintes seções, refletindo a estrutura de navegação do aplicativo:

- **Visão Geral (Dashboard):** Uma introdução à tela inicial, seus principais indicadores e funcionalidades.
- **Cotações:** Um guia detalhado sobre como gerenciar, criar e acompanhar o status das cotações.
- **Cadastros:** Instruções sobre como gerenciar as entidades fundamentais do sistema, incluindo colaboradores, seguradoras, contatos e corretoras.
- **Automação:** Uma explicação sobre como configurar e utilizar as automações para otimizar processos.
- **E-mail (Templates):** Um guia sobre como gerenciar e utilizar templates de e-mail para comunicação padronizada.

Utilize o menu de navegação à esquerda para explorar os tópicos de seu interesse.
