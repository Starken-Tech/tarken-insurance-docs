# 4. Automação

A seção de **Automação** é uma das funcionalidades mais poderosas da plataforma Starken Insurance. Ela permite a criação de regras e gatilhos que automatizam tarefas repetitivas, otimizam o fluxo de trabalho e garantem que os processos sejam seguidos de maneira consistente.

#### Estrutura da Página

A página de automações lista todas as regras ativas e inativas e oferece ferramentas para seu gerenciamento.

* **Barra de Ferramentas:**
  * **Nova automação:** Botão para criar uma nova regra de automação.
  * **Filtrar e Ordenar:** Ferramentas para localizar e organizar as automações existentes.

#### Tabela de Automações

A lista de automações é exibida em uma tabela detalhada:

| Coluna         | Descrição                                                                         |
| -------------- | --------------------------------------------------------------------------------- |
| **Automação**  | O nome ou código que identifica a regra de automação.                             |
| **Evento**     | O gatilho que aciona a automação (ex: "Cotação alterada", "Oportunidade criada"). |
| **Produtos**   | Ícones que indicam a quais produtos de seguro a automação se aplica.              |
| **Status**     | Mostra se a automação está **Ativa** ou **Inativa**.                              |
| **Modificado** | A data da última alteração na regra.                                              |
| **Criado**     | A data em que a regra foi criada.                                                 |

#### Exemplos de Automação

O sistema permite configurar uma vasta gama de automações para diferentes cenários, como:

* **Criação de Atividades:** Criar tarefas automaticamente quando uma cotação avança para uma nova etapa (ex: "Criar Atividade de Negociações" quando o status muda para "Em análise").
* **Gestão de Status:** Alterar o status de uma cotação com base em eventos específicos (ex: "Forçar Cotação Declinada/Perdida" quando um gatilho de declínio é acionado).
* **Reabertura de Processos:** Automatizar a reabertura de atividades quando um processo que havia sido declinado é retomado.
* **Conclusão de Etapas:** Marcar etapas como concluídas automaticamente após a finalização de tarefas-chave (ex: "Concluir Etapas do Pós Venda").

O uso inteligente de automações reduz significativamente o trabalho manual, minimiza a ocorrência de erros e garante que os processos de cotação e pós-venda sejam executados de forma padronizada e eficiente.
