# 3. Cadastros

A seção de **Cadastros** é fundamental para o funcionamento da plataforma Starken Insurance, pois é onde são gerenciadas todas as entidades que interagem no ecossistema de seguros. Manter esses dados organizados e atualizados é crucial para garantir a eficiência dos processos de cotação, comunicação e automação.

### Estrutura da Seção

Esta seção agrupa o gerenciamento de quatro tipos principais de entidades:

* **Colaboradores:** Usuários internos da plataforma, como administradores, analistas e gerentes.
* **Seguradoras:** As companhias de seguro parceiras que oferecem os produtos.
* **Contatos:** Pessoas de referência dentro das seguradoras, com quem a equipe interage.
* **Corretoras:** As corretoras de seguro parceiras que utilizam a plataforma para gerenciar suas cotações.
