# 2.2. Criando uma Nova Cotação

O processo de criação de uma nova cotação na plataforma Starken Insurance é iniciado através de um formulário em etapas, acessado pelo botão **"Criar cotação"**. Este formulário guia o usuário de forma intuitiva, garantindo que todas as informações necessárias sejam coletadas de maneira estruturada.

#### Etapa 1: Seleção de Produto e Responsável

Ao iniciar uma nova cotação, a primeira etapa do formulário solicita informações essenciais para direcionar o processo.

**Informações do Produto**

Nesta seção, o usuário deve definir o escopo da cotação:

* **Produto:** Um menu suspenso (dropdown) para selecionar o tipo de seguro desejado (ex: RC Ambiental, Garantia, etc.).
* **Ramo:** Um segundo menu suspenso que permite especificar o ramo do seguro, dependendo do produto selecionado.

**Informações de Preenchimento**

Aqui, o usuário define quem será o responsável por fornecer os detalhes da cotação:

* **Corretora:** Seleção da corretora que está intermediando a negociação a partir de uma lista de parceiros cadastrados.
* **Responsável pelo preenchimento:** O usuário pode escolher entre duas opções:
  * **Corretor:** O próprio corretor preencherá o Questionário de Avaliação de Risco (QAR).
  * **Segurado:** Um link será enviado para que o cliente final preencha o questionário diretamente.

#### Próximas Etapas

Após preencher as informações iniciais, o usuário clica em **"Avançar"** para prosseguir para as próximas fases do formulário, que podem incluir:

* Preenchimento detalhado do Questionário de Avaliação de Risco (QAR).
* Anexo de documentos relevantes.
* Definição de coberturas e limites.

O botão **"Cancelar"** está disponível a qualquer momento para descartar o processo de criação da cotação.
