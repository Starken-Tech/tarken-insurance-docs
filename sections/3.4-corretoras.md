# 3.4. Corretoras

A seção **Corretoras** destina-se ao gerenciamento das corretoras de seguros parceiras que utilizam a plataforma Starken para gerenciar suas cotações. Este cadastro é essencial para a correta atribuição de cotações e para o controle de acesso e permissões.

#### Estrutura da Página

A interface permite uma fácil visualização e gerenciamento das corretoras cadastradas.

* **Barra de Ferramentas:**
  * **Criar cadastro:** Botão para registrar uma nova corretora parceira.
  * **Filtrar e Ordenar:** Ferramentas para organizar e localizar corretoras na lista.

#### Tabela de Corretoras

A lista de corretoras é exibida em uma tabela que resume as informações principais:

| Coluna        | Descrição                                                                                                                                        |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Corretora** | O nome e o logotipo da corretora.                                                                                                                |
| **ID**        | Um identificador numérico único para cada corretora no sistema.                                                                                  |
| **Cidade**    | A localização geográfica da sede ou principal filial da corretora.                                                                               |
| **Produtos**  | Ícones que representam os produtos de seguro que a corretora está habilitada a comercializar. Um contador (ex: "+1") indica produtos adicionais. |

O gerenciamento adequado das corretoras garante que cada parceiro tenha acesso apenas às suas próprias cotações e informações, mantendo a integridade e a segurança dos dados na plataforma.
