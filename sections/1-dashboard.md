# 1. Visão Geral (Dashboard)

O **Dashboard** é a página inicial do aplicativo Starken Insurance, projetada para oferecer uma visão geral e consolidada das atividades mais importantes da plataforma. Ele serve como um ponto de partida central para o usuário, apresentando indicadores-chave de desempenho (KPIs), atalhos para ações rápidas e um resumo das cotações recentes.

### Estrutura da Página

A tela do Dashboard é dividida em três áreas principais:

1. **Cabeçalho Superior:** Contém elementos de navegação e identidade visual.
2. **Área de Ações Rápidas:** Botões para as funcionalidades mais utilizadas.
3. **Widgets e Gráficos:** Painéis interativos com dados e métricas em tempo real.

#### Elementos do Cabeçalho

O cabeçalho superior oferece acesso rápido a funcionalidades globais:

* **Logo Starken:** Localizado no canto superior esquerdo, serve como identidade visual da plataforma.
* **Barra de Pesquisa:** Permite buscar rapidamente por informações específicas, como o nome de um segurado ou corretora.
* **Ícone de Notificações:** Exibe alertas e atualizações importantes sobre as atividades do usuário.
* **Menu do Usuário:** Mostra o nome e a foto do usuário logado, com acesso a configurações de perfil e à opção de sair do sistema (logout).

#### Funcionalidades Principais

Logo abaixo do cabeçalho de boas-vindas, a área de ações rápidas inclui:

* **Botão "Criar cotação":** Atalho principal para iniciar um novo processo de cotação, uma das ações mais frequentes na plataforma.
* **Botão "Filtrar":** Permite aplicar filtros aos dados exibidos nos widgets do Dashboard, como período de análise.
* **Botão "Largura total":** Oferece a opção de expandir a visualização dos painéis para ocupar toda a largura da tela, facilitando a análise de dados.

### Widgets e Gráficos

O corpo do Dashboard é composto por widgets que apresentam dados de forma visual e intuitiva.

#### Status das Negociações

Este widget exibe um resumo quantitativo e percentual das negociações em andamento, distribuídas de acordo com seu status no funil de vendas. Os status incluem:

* Aguardando QAR (Questionário de Avaliação de Risco)
* Validação
* Em análise
* Negociação
* Assinaturas
* Em emissão
* Declinado
* Concluídos

#### Gráficos de Desempenho

O Dashboard inclui gráficos que ajudam a visualizar a performance de diferentes áreas:

| Gráfico         | Descrição                                                                                                                                                                |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Produtos**    | Apresenta a distribuição de cotações por tipo de produto (ex: RC Ambiental, Transporte), permitindo identificar quais produtos têm maior demanda.                        |
| **Corretoras**  | Mostra o volume de cotações por corretora, oferecendo insights sobre o desempenho de cada parceiro. Inclui opções para exportar os dados em formato **SVG, PNG ou CSV**. |
| **Seguradoras** | Exibe a performance por seguradora. Caso não haja dados no período filtrado, uma mensagem informativa é exibida.                                                         |

#### Cotações Recentes

Uma tabela na parte inferior da página lista as cotações mais recentes, fornecendo acesso rápido aos processos em andamento. As colunas exibidas são:

* **Data da solicitação**
* **Produto**
* **Segurado**
* **Status**

Esta tabela permite que o usuário acompanhe as últimas atualizações sem precisar navegar para a seção de cotações.
