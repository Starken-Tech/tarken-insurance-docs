> ## 3.3. Contatos

A seção **Contatos** funciona como uma agenda centralizada, armazenando as informações de contato dos profissionais que atuam nas seguradoras parceiras. Manter esta base de dados atualizada é vital para agilizar a comunicação durante o processo de cotação e negociação.

### Estrutura da Página

A página de contatos foi projetada para ser uma ferramenta de consulta rápida e eficiente.

-   **Barra de Ferramentas:**
    -   **Pesquisar:** Um campo de busca para localizar rapidamente um contato.
    -   **FILTRAR:** Botão para aplicar filtros avançados à lista, com um contador que indica o número de contatos visíveis.
    -   **CADASTRAR:** Permite adicionar um novo contato à agenda.

### Tabela de Contatos

A lista de contatos é organizada em uma tabela clara e informativa:

| Coluna       | Descrição                                                                                                 |
|--------------|-----------------------------------------------------------------------------------------------------------|
| **Seguradora** | Indica a companhia de seguros à qual o contato está vinculado.                                              |
| **Contato**    | O nome do profissional.                                                                                   |
| **E-mail**     | O endereço de e-mail principal para comunicação.                                                          |
| **Whatsapp**   | O número de telefone para contato via WhatsApp.                                                           |
| **Telefone**   | O número de telefone fixo ou alternativo.                                                                   |
| **Produtos**   | Ícones que representam os produtos de seguro com os quais o contato trabalha.                             |
| **Ações**      | Um menu de opções (representado por três pontos verticais) que permite editar ou excluir o contato.       |

Um cadastro de contatos bem estruturado garante que a equipe da Starken possa interagir com as pessoas certas dentro de cada seguradora, otimizando o tempo de resposta e fortalecendo o relacionamento com os parceiros.
