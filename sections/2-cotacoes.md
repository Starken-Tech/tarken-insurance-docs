> # 2. Cotações

A seção de **Cotações** é o coração da plataforma Starken Insurance, onde os usuários podem gerenciar todo o ciclo de vida das cotações de seguros. Esta área foi projetada para oferecer uma visão clara e organizada do funil de vendas, desde a solicitação inicial até a emissão da apólice.

## Estrutura da Seção

Esta seção é dividida em duas partes principais para facilitar a navegação e o gerenciamento:

-   **Gerenciando Cotações (Kanban):** Uma visão geral do fluxo de trabalho das cotações, organizada em um painel Kanban intuitivo.
-   **Criando uma Nova Cotação:** O processo passo a passo para iniciar uma nova cotação na plataforma.
