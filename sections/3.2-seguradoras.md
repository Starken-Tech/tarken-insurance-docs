# 3.2. Seguradoras

A seção **Seguradoras** é dedicada ao gerenciamento das companhias de seguro parceiras cadastradas na plataforma. Manter esta lista atualizada é essencial para associar corretamente os produtos de seguro e os contatos comerciais.

#### Estrutura da Página

A interface desta seção é composta por uma barra de ferramentas e uma tabela que lista todas as seguradoras cadastradas.

* **Barra de Ferramentas:**
  * **Criar cadastro:** Permite adicionar uma nova seguradora à plataforma.
  * **Filtrar e Ordenar:** Ferramentas para facilitar a busca e a organização da lista de seguradoras.

#### Tabela de Seguradoras

A tabela exibe informações importantes sobre cada seguradora:

| Coluna         | Descrição                                                                                                                                       |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| **Seguradora** | O nome e o logotipo da companhia de seguros.                                                                                                    |
| **ID**         | Um identificador numérico único para cada seguradora no sistema.                                                                                |
| **Produtos**   | Ícones que representam os diferentes produtos de seguro oferecidos pela seguradora. Um contador numérico (ex: "+1") indica produtos adicionais. |

O cadastro correto das seguradoras é um passo fundamental para o bom funcionamento do fluxo de cotações, pois garante que as propostas sejam direcionadas às companhias certas e que os produtos estejam corretamente associados.
